+ (String) generate(assigns = {}) { ... }

its goal is to generate function calls.

Examples:

generate { jquery.jq! } # => "jQuery.jQuery()"
generate { a(1 => "a", :b => [1,2]) # => "a({\"1\":\"a\",\"b\":[1,2]})"

generate { b(function(b!, :a) {b!.c!} } # => "b(function(b(), "a") {b().c();});"
