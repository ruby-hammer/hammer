- (Object) define_widget(name = :Widget, parent = nil) { ... }

defines widget and executes #extend_widget for hooks

Examples:

widget with name Header 

define_widget :Header do
  wrap_in :h1
  def content
    text "A header"
  end
end

quickly defined widget with default name Widget 

define_widget :quickly do
  def content
    h1 "A header"
  end
end

Parameters:

  • (Symbol) name (defaults to: :Widget) —

    of a new widget class. If :quicly is passed defines widget quickly which means that block is used to define #content not class

  • (Symbol, Class) parent (defaults to: nil) —

    class of new widget class. Symbol is used to find widget class with same name in parent components. Class is used directly. see #parent_widget_class

Yields:

  • block which is evaluated inside new widget class or #contet if :quicly is used

def define_widget(name = :Widget, parent = nil, &block)
  if name == :quickly
    define_widget { define_method :content, &block }
  else
    parent = parent_widget_class(parent || name)
    widget_classes[name] = widget_class = const_set(name, Class.new(parent))
    extend_widget(widget_class)
    widget_class.class_eval(&block)
  end
end

def parent_widget_class(name)
  return name if name.kind_of? Class
  check_class widget_classes[name] || superclass.try(:parent_widget_class, name)
end
